# rms

Rename My Schema

```
RMS (Rename My Schema) Version: v0.0.1
Usage of rms:
  -from string
        from schema name
  -host string
        host (default "localhost")
  -password string
        password
  -port int
        port (default 3306)
  -table value
        table to be migrated
  -to string
        to schema name
  -user string
        username
```
